## Usage

### Clone the repositorie
```console
$ https://github.com/Epic-R-R/YoutubeDL_Bot.git
```
### Go to directory and run project
```console
$ cd tTodo
```
### Token BOT
>* *Open the main file with the text editor and place your robot token in the YOUR_TOKEN_BOT location.*

### Run project
```console
$ python main.py
```
